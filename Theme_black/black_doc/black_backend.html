Back-end

What is Back-end?

  백엔드는 웹 사이트나 웹 애플리케이션 또는 모바일 솔루션의 프로세스와 관련된 서버 측(Server-side)과 데이터베이스를 관리해주는 테크놀로지이다. 프론트엔드의 반대 말로, 백엔드는 눈에 보이지 않는 서버에서 작용하는 기술을 다룬다.

Back-end Development

  백엔더는 서버 측 개발 분야로, 데이터를 저장하고 관리하며, 웹사이트의 클라이언트 측에서 모든 것이 매끄럽게 작동할 수 있게 해준다. 백엔드는 사용자들이 직접 보거나 상호작용할 수는 없지만, 프론트엔드에 있는 사용자들이 취하는 행동들을 처리하는 역할을 수행한다. 즉 사용자들은 백엔드에 직접적으로가 아닌, 간접적으로 백엔드에 엑세스하게 되는 것이다. 백엔드 개발에는 사용자 인터페이스가 없는 시스템 컴퓨넌트 작업, API 작성, 라이브러리 생성, 데이터베이스 통합 등 다양한 활동들이 포함되어 있다.


Programming Language

  프론트엔드에 비해 백엔드는 선택의 폭이 상당히 넓다고 할 수 있다. 어떤 언어라도 개발이 충분히 가능한 편이기에 언어에 개발이 크게 좌우되는 편이 아니다. 그렇기에 백엔드에서는 해당 언어가 가진 특성과 작업물 및 작업 방향과의 일치성 등을 고려하여 선택하는 것이 좋다. 기업에서도 다양한 언어와 프레임워크들이 백엔드 개발에 사용되고 있다.


PHP

  PHP(Hypertext Preprocessor)는 개발을 위해서 고안된 서버 측 스크립트 언어이자 수많은 웹 시스템의 기반이 되는 언어이다. 우리나라에서도 다수의 인터넷 쇼핑몰, 커뮤니티 등이 PHP거나 PHP 기반으로 제작되어 있다. PHP는 브라우저에서 요청한 웹 페이지, 즉 서버에 있는 데이터가 사용자와 상호작용하는 웹 어플리케이션으로 진화하여 브라우저로 넘어가도록 처리하는 데 필요하다. PHP로 작성된 코드를 HTML 코드 안에 추가하면 웹 서버는 해당 코드를 해석하여 자동으로 HTML 문서를 생성하기에 동적으로 빠르게 HTML 문서를 만들 수 있다. 이처럼 PHP는 HTML을 프로그래밍적으로 생성해 주고, 데이터베이스와 상호작용 하면서 데이터를 저장하고, 표현하는 서버와 브라우저 간의 다리 역할을 한다고 볼 수도 있다.

Java

  Java는 가장 인기 있는 프로그래밍 언어 중 하나이다. 객체지향 프로그래밍 언어인 자바의 가장 큰 특징은 플랫폼 종속성이 낮다는 것이다. 운영체제에 Java Virtual Machine(JVM)을 설치하면 Java 기반 프로그램은 어떤 환경에서도 완벽하게 똑같이 동작한다. 이유로는 가상머신이 각각의 운영체제에 맞춰서 돌아갈 수 있도록 제작되었기 때문이다. 프론트엔드에서 JavaScript는 HTML과 CSS가 존재한다는 가정하에 실행되었는데, Java는 특정 플랫폼에 의존하지 않는 독립성을 갖추고 있다. 생산성과 안정성, 기기 호환성이 높고 소스코드 가독성이 좋아 개발자 커뮤니티에서 널리 사용되고 있다. 특히 Java는 Spring이라는 성능이 매우 좋은 웹 프레임워크를 지원한다. 이 Spring은 한국 전자정부표준프레임워크의 기반기술이며 한국정보화진흥원에서는 공공이관의 웹 서비스 제공시 Spring을 권장할 정도로 인지도가 높은 프레임워크이기에 한국에서 Java는 백엔드에서 많은 사랑받고 있다.

Node.js

  JavaScript는 Node.js라는 크로스 플랫폼의 오픈소스 런타임 환경 덕분에 백엔드 서비스 개발에도 사용할 수 있게 되었다. Node는 브라우저 외부에서 JavaScript를 실행할 수 있게 하여 JavaScript가 웹 브라우저를 벗어나 모바일이나 웹 애플리케이션용 API와 같은 백엔드 서비스 개발에도 사용되는 범용 프로그래밍 언어가 될 수 있게 되었다. Node.js는 내장 HTTP 서버 라이브러리를 포함하고 있어 웹 서버에서 별도의 소프트웨어 없이 동작하는 것이 가능하다. 확장성이 뛰어나고 반응속도가 빠르며, 많은 양의 요청도 처리 가능하기에 웹 서버의 동작을 더욱 효과적으로 통제할 수 있다. 하지만 언어가 아니라 소프트웨어 플랫폼이어서 그 자체로는 아무것도 못하고 서버를 직접 구현해야 한다는 특징을 가지고 있다.

Python

  Python은 개발자들이 시스템을 효율적으로 통합하고, 빠르게 작업할 수 있게 해주는 프로그래밍 언어이다. 파이썬은 딥러닝, 데이터 사이언스, 인공지능 분야에서 많이 사용되는데 Django 혹은 Flask와 같은 웹 프레임워크랑 사용되어 백엔드 서버를 구현하는데 사용되기도 한다. 실제로 Python을 백엔드 언어로 사용 중인 기업으로는 Instagram, 번개장터, Reddit, Pinterest 등이 있다.
